# Mods

This page is where you install, enable, disable, and remove mods for your server.

![Full Mods page](../images/wiki/mods-01-overview.png)
*(Screenshot placeholder - a full view of the Mods page)*

## How do I browse for new mods?

Click **Browse Nexus Mods** at the top right of the page. This opens a search window where you can look through Palworld mods on Nexus Mods - no account or API key needed just to browse.

![Browse Nexus Mods button](../images/wiki/mods-02-browse-button.png)
*(Screenshot placeholder - circle the "Browse Nexus Mods" button)*

## How do I install a mod I found?

In the browse window, each mod has an install option:

- If the super admin has connected a Nexus Premium API key, you'll see **Install** - one click and it's installed.
- If not, you'll see **Add to Wishlist** instead. This sends a request to the super admin, who can approve it from the [Mod Wishlist](mod-wishlist.md) page.

![Install or Add to Wishlist button in the browse dialog](../images/wiki/mods-03-install-or-wishlist.png)
*(Screenshot placeholder - circle the Install / Add to Wishlist button on a mod card in the browse window)*

If a mod has more than one file to choose from (like a Main File and some Optional Files), you'll see a small picker so you can choose exactly which one you want.

![File picker for a mod with multiple files](../images/wiki/mods-04-file-picker.png)
*(Screenshot placeholder - the file picker dialog, if you have a mod with multiple files to show)*

## How do I turn a mod on or off?

Every installed mod has a toggle switch on its card. Flip it to enable or disable that mod without removing it.

![Mod enable/disable toggle](../images/wiki/mods-05-toggle.png)
*(Screenshot placeholder - circle the on/off toggle on a mod card)*

## How do I change which mods load first?

Each mod card has a drag handle. Click and drag it up or down to change that mod's load priority.

![Drag handle for reordering mods](../images/wiki/mods-06-drag-handle.png)
*(Screenshot placeholder - circle the drag handle icon on a mod card)*

## How do I update or remove a mod?

If an update is available, an **Update** button appears on that mod's card. To remove a mod entirely, use the **Remove** button and confirm in the popup.

![Update and Remove buttons on a mod card](../images/wiki/mods-07-update-remove.png)
*(Screenshot placeholder - circle the Update and Remove buttons on a mod card)*

## How do I install UE4SS?

Scroll to the bottom of the page for the UE4SS panel. This is required for most Lua-based mods to work.

![UE4SS install panel](../images/wiki/mods-08-ue4ss.png)
*(Screenshot placeholder - the UE4SS panel and its install button)*

> Restart the server any time you change mods, so Palworld picks up the change.
