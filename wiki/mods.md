# Mods

Manage every mod installed on the active server, and browse Nexus Mods for new ones, without ever leaving the browser.

![Mods screenshot placeholder](../images/wiki/mods.png)

## What you can do here

- See every installed mod, along with its enabled/disabled/broken status.
- Enable, disable, or remove a mod.
- Drag mods to reorder them - this controls Palworld's mod load priority.
- Check for a mod update and install it in one click.
- Browse Palworld mods on Nexus Mods (no personal Nexus account or API key required just to browse).
- Install a mod directly from Nexus if the super admin has connected a Premium API key, or add it to the [Mod Wishlist](mod-wishlist.md) for the super admin to approve otherwise.
- Install and update UE4SS from the panel at the bottom of this page.

## Notes

- If Nexus mod files with more than one download (e.g. a Main File plus Optional Files) are installed directly, you'll get a picker to choose exactly which file you want.
- Verified manual file installs (uploading a file you downloaded yourself) happen from [Super Admin](super-admin.md), and check the uploaded file's exact hash against Nexus's own catalog before installing.
